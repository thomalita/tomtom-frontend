### **_Description_**

A Webapp that brings some of the most popular Anime Characters from shows many people love. Will show a detail page of their story and will include pictures.  

---

### **_Technologies Used_**

#### PERN Stack

- React
- Postgres
- HTML
- Node.js
- JavaScript
- CSS

### **_Getting Started_**

- cd client
- npm install
- npm install react-router-dom
- npm install axios
- npm run start
